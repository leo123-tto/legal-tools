# 中国高速自驾游 - 竖屏版

一款竖屏高速公路驾驶模拟游戏，玩家可以驾驶车辆在中国高速公路网中行驶，探索全国各个城市。

## 游戏特点

- **竖屏适配**：专为移动端设计，适合手机操作
- **Canvas绘制背景**：道路、服务区全部通过Canvas实时绘制，支持无限滚动
- **滑动控制**：全屏滑动控制车辆上下左右移动，操作简便
- **真实场景**：服务区使用真实名称，如梅村服务区、阳澄湖服务区等
- **答题系统**：在服务区答题可以加油补给
- **丰富题库**：包含口算、拼音、车牌知识、高速知识、地理知识、交通常识等180+题目
- **路网探索**：岔道选择前往不同城市，右下角显示当前高速公路名称
- **时间天气**：时间随里程变化，光线效果跟随变换；天气周期性变化
- **积分榜**：记录并显示单次最高积分

## 游戏操作

- **滑动控制**：在屏幕上滑动控制车辆上下左右移动
- **键盘控制**：PC端可以使用方向键或WASD键

## 游戏规则

1. **目标**：驾驶车辆在高速公路上行驶，探索全国路网
2. **服务区**：道路右侧出现服务区建筑时，将车移到右侧车道即可触发答题加油
3. **岔道**：行驶过程中出现岔道选择，选择不同方向前往不同城市
4. **躲避车辆**：注意前方车辆，及时变道躲避
5. **结束条件**：油量耗尽或撞车则游戏结束

## 游戏系统

### 时间变换
- 每行驶200公里变换时段
- 时段循环：清晨 → 上午 → 中午 → 下午 → 黄昏 → 夜晚
- 光线效果跟随变换

### 天气变换
- 每行驶300公里变换天气
- 天气类型：晴、阴、雨、雪

### 高速公路信息
- 右下角显示当前高速公路编号和名称
- 选择岔道时自动更新
