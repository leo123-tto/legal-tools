# 中国高速自驾游 - 竖屏版

一款竖屏高速公路驾驶模拟游戏，玩家可以驾驶车辆在中国高速公路网中行驶，探索全国各个城市。

## 最新版本

v2.3（2026-04-10）

## 游戏特点

- **竖屏适配**：专为移动端设计，适合手机操作
- **Canvas绘制背景**：道路、服务区全部通过Canvas实时绘制，支持无限滚动
- **双操作模式**：支持滑动控制 + 点击变道，PC端支持键盘WASD/方向键
- **真实路网**：服务区按实际高速公路顺序出现，数据来源于真实路网
- **答题系统**：在服务区答题可以加油补给
- **丰富题库**：包含口算、拼音、车牌知识、高速知识等180+题目
- **路网探索**：岔道选择前往不同城市，右下角显示当前高速公路名称
- **时间天气**：时间/天气随里程变化，有提示动画
- **积分榜**：记录并显示单次最高积分

## 游戏操作

- **点击变道**：点击屏幕左侧往左变道，右侧往右变道（推荐）
- **滑动控制**：在屏幕上滑动控制车辆上下左右移动
- **键盘控制**：PC端可以使用方向键或WASD键

## 游戏规则

1. **目标**：驾驶车辆在高速公路上行驶，探索全国路网
2. **服务区**：道路右侧出现服务区建筑时，将车移到右侧车道即可触发答题加油
3. **岔道**：行驶过程中出现岔道选择，选择不同方向前往不同城市
4. **躲避车辆**：碰撞会扣分扣油（不结束游戏）
5. **结束条件**：油量耗尽则游戏结束

## 游戏系统

### 真实路网数据

| 高速编号 | 名称 | 路线 | 服务区数量 |
|---------|------|------|-----------|
| G42 | 沪蓉高速 | 上海↔成都 | 15+ |
| G2 | 京沪高速 | 北京↔上海 | 25+ |
| G25 | 长深高速 | 南京↔杭州 | 7 |
| S48 | 锡宜高速 | 无锡↔宜兴 | 2 |

### 时间变换

- 每行驶100公里变换时段
- 时段循环：清晨 → 上午 → 中午 → 下午 → 黄昏 → 夜晚
- 有提示动画

### 天气变换

- 每行驶150公里变换天气
- 天气类型：晴、阴、雨、雪
- 有提示动画

### 碰撞系统

- 碰撞不再结束游戏
- 扣10分 + 扣5升油 + 2秒无敌时间
- 有碰撞提示动画

### 操作优化

- 点击屏幕左/右侧可直接变道
- 滑动控制保留备选
- 键盘支持WASD和方向键（长按连续移动）

## 本地运行

```bash
cd highway-drive-game-vertical
python3 -m http.server 8081
```

然后在浏览器中访问 `http://localhost:8081`
