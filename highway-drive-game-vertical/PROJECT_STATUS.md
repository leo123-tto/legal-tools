## 核心游戏机制

### 道路系统
- 道路宽度：屏幕宽度的55%
- 道路位置：靠左显示，左边距20px
- 三车道设计
- 黄色边缘线 + 白色虚线分隔线

### 控制方式
- **滑动控制**：全屏滑动控制车辆上下左右移动
- **键盘控制**：PC端可使用方向键或WASD键

### 服务区
- 位置：道路右侧
- 触发：玩家在右侧车道时自动触发答题
- 加油：答对题目加20升油

### 时间变换
- 每200公里变换时段
- 时段循环：清晨 → 上午 → 中午 → 下午 → 黄昏 → 夜晚
- 光线效果跟随变换

### 天气变换
- 每300公里变换天气
- 天气类型：晴、阴、雨、雪
